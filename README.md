# NoSQLProvider
A cross-browser/platform indexeddb-like client library

Documentation will be forthcoming.  For now, I would suggest not using this yet, as it requires some more work.

## Compling
### Source
```bash
npm install
npm run build
```
### Tests
```bash
npm install
npm run webtest
```

## Testing
1. Compile tests
1. Open test.html in browser
